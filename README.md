# My-Mother-Day
献给妈妈的母亲节网页 - 一份充满爱与温暖的数字礼物
## 🎀 项目简介
这是一个为母亲节精心制作的单页纪念网站，使用纯 HTML、CSS 和 JavaScript 构建。网站以水墨晕染的视觉风格为主调，融合了时间轴、照片相册、温馨寄语等元素，向妈妈表达最深的爱与感恩。
### 网页须知
使用全新TRAE SOLO网页端进行全站开发，人物图片均为AI生成，如需使用请下载源文件进行修改即可使用
